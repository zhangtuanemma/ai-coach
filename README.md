# 问题处理 AI 教练

帮助站长更好地处理客户和员工问题的智能教练系统。

## 核心理念

### 三大心态转变
1. **我是公司，我是老板** - 不是传话筒，是负责人
2. **问题不是麻烦，是机会** - 拥抱问题，从中成长
3. **不怕吃亏，才能做好服务** - 以对方利益为出发点

### 沟通四步法
1. **听** - 让对方说完，认真倾听
2. **认** - 找到对方对的地方，真诚认可
3. **问** - 换位思考，引导对方思考
4. **定** - 给方案+定时间，承诺兑现

## 使用方法

### 方式1：使用 AI 教练程序（推荐）

运行交互式教练程序：

```bash
python3 ai_coach.py
```

程序会通过提问引导你：
1. 描述问题
2. 检查心态
3. 学习沟通方法
4. 参考相关案例
5. 生成反思总结

所有记录会自动保存到 `问题处理记录.txt`

### 方式2：使用工具模板

使用 `问题处理工具.md` 中的检查清单：

1. **第一步：检查心态**
   - 心态1：我是公司，我是老板
   - 心态2：问题不是麻烦，是机会
   - 心态3：不怕吃亏，才能做好服务
   - 沟通心态：了解事实 vs 说服对方

2. **第二步：沟通四步法**
   - 听：倾听对方
   - 认：认同对方
   - 问：换位思考
   - 定：给方案+定时间

3. **第三步：填写汇报表**
   - 事件准备
   - 问题识别
   - 改进计划

## 常见问题

### Q: 我觉得客户很难搞，怎么办？
A: 转变心态。挑剔的客户是磨刀石，能让你成长。客户身边都是客户，服务好了能带来更多客户。

### Q: 我怕吃亏，怎么办？
A: 不怕吃亏，才能做好服务。人的一生中要做一些亏钱的生意。适当的退让能换取信任。

### Q: 我不知道怎么跟客户沟通，怎么办？
A: 使用沟通四步法：听、认、问、定。先听完，再认同，然后换位思考，最后给方案定时间。

### Q: 我觉得这不是我的问题，是公司的问题，怎么办？
A: 转变角色。对员工来说，你就是公司。不要当传话筒，要当负责人。

### Q: 我处理问题总是很急，怎么办？
A: 不要太急于处理一件事。如果你觉得特别难搞，你一定要想好策略再去做。心态够好，事情就不复杂。
